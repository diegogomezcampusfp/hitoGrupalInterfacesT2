# Proyecto

Para mostrar un proyecto con 5 vistas el cual tenga buena estructura y detalles, hemos decidido utilizar la plantilla Material Dashboard Angular, la cual nos permite crear nuestro propio proyecto teniendo de base una plantilla. Esta plantilla cuenta con 7 vistas.


## Comandos de la consola

Este proyecto fue generado con Angular CLI versión 1.0.0 y Angular 4.x.

Instale NodeJs desde la Página Oficial de NodeJs.
Abra la terminal.
Vaya a la carpeta de su proyecto.
Asegúrese de haber instalado Angular CLI previamente. Si no lo ha hecho, por favor instálelo.
Ejecute en la terminal: npm install
Ejecute ng serve para iniciar un servidor de desarrollo. Vaya a http://localhost:4200/. La aplicación se recargará automáticamente si cambia alguno de los archivos de origen.
